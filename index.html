CSS Grid Polyfill

Шаг 1

Создаем грид-контейнер

        .grid-container {
  display: grid;
}

или

          .grid-container {
  display: inline-grid;
}

Шаг 2

Рисуем виртуальную сетку

          .grid-container {
  display: grid;
  grid-template-columns: 250px 1fr 100px;
  grid-template-rows: 80px 80px 80px;
}

Шaг 3

Задаем шаблон областей

  display: grid;
  grid-template-rows: auto 1fr auto;
  grid-template-areas: "header"

                       "footer";
}

Классно, правда?

  • интуитивно понятно
  • легкая адаптация с помощью media-запросов
  • порядок элементов в разметке может быть любой
